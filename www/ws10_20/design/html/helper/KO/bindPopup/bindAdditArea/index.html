Bind Additional Info Area

Binding 되어 있는 Field의 Binding 추가 Option을 설정을 할 수 있게 한다
1. additional Options 설명
  • Field name : 현재 바인딩 하기 위해서 선택한 필드명
  • Field path : 선택한 필드가 위치하고 있는 경로
  • type : 선택한 필드가 참조하고 있는 Type
  • Bind type : 숫자 타입의 필드에 적용 시킬 단위 타입 선택
  • ( sap.ui.model.type.Currency / ext.ui.model.type.Quantity )
  • Reference Field name : Bind Type 설정 시 참조 할 필드 명
  • Conversion Routine : 선택한 필드에 Conversion Rotuine 적용
  • Nozero : 숫자 타입일 경우 Initial 값으로 0을 보이게 할지 여부 결정
  • Is number fomat? : 숫자 타입의 필드일 경우 숫자 포맷으로 보이게 할지 여부 결정
2. Model Tree, Design Tree에서 선택된 Field에 따라 Reference Field이 구성된다.

Model Field Area의 Tree에서 선택된 필드에 따른 Refrence Field 예시


> Model Field Area의 Tree에서 선택된 'GT_OTAB'의 필드에서는 Currecy Bind Type에 참조 할 수 있는 필드들이 존재하지만,
'GS_HDL'에서 선택된 필드들은 참조할 수 있는 필드들이 없어 리스트가 나오지 않는다.

'GS_HDL'에서 선택된 필드들은 참조할 수 있는 필드들이 없어 리스트가 나오지 않는다.
