Apply Equal Properties Area

Design Area에서 선택한 UI와 동일한 속성의 UI들의 Binding 동기화 적용한다
1. Design Area에서 동기화 적용할 UI 선택 후 버튼클릭하면 Apply Equal Properties Area로 전환됨

Apply Equal Properties Area에서는 추가 속성 적용 버튼 비활성화된다

※ Design Area에서 Binding 동기화를 하기 위해 Apply Equal Properties로 이동 불가능한 경우


① 선택한 Property에서 Binding된 필드가 없음


② 선택한 Property가 없음


③ Property를 1개가 아닌 N건 선택함


④ Property가 선택한 UI 이외에 존재하지 않음

2. Apply Equal Properties Area에서 동기화
- 선택한 UI를 기준으로 동기화 할 UI들을 선택후 버튼을 클릭하여 Binding 동기화

'TITLE1'에 바인딩한 'GS_ADD-BRGEW' 필드가 선택한 'TEXT300' UI의 text property에 바인딩되고 추가속성 정보까지 동기화됨을 확인

3. Synchronizion Equality popup에서의 Binding 동기화
3.1 Synchronizion Equality popup Open
※ UI 미 선택후 Apply all 버튼 클릭 시, Error
3.2 선택한 UI를 기준으로 동기화 할 UI들을 선택후 버튼을 클릭하여 Binding 동기화
4. Apply Equal Properties Area 구성 버튼기능, 영역 추가 설명
  • ① 선택한 UI의 속성, 바인딩 정보 영역
  • ② Binding 추가 속성 영역
  • ③ 칼럼 최적화
  • ④ Design Area로 돌아가는 Back 버튼
